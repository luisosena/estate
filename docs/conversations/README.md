# Conversation Logs

This directory contains permanent records of important conversations and decisions.

## Structure
- `YYYY-MM-DD-conversation-topic.md` - Individual conversation logs
- `decisions.md` - Key project decisions
- `patterns.md` - Reusable code patterns and solutions

## Usage
Each conversation log includes:
- Date and timestamp
- Topic summary
- Key decisions made
- Code solutions implemented
- Follow-up actions needed
